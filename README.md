# 🎬 StreamFlix Movie Explorer

StreamFlix Movie Explorer is a simple movie browsing website built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 🔎 Movie search functionality
* 🎥 Watch movie trailers
* ⭐ User rating system
* 📽 Movies from Hollywood, Bollywood, and Kollywood
* 🎬 Trailer popup modal

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
